# 實況主斗內平台
[`https://nekolive.net/`](https://nekolive.net/)

方便實況主們使用第三方金流（歐付寶、藍新、統一、Paypal、綠界）收取斗內款項，平台提供 Twitch 機器人，讓實況更有互動性，另外也提供許多功能：加班計時器、斗內列表、進度條、媒體點播。  

網站設立於 2023/04 至今，由 [猫又おにぎり](https://github.com/haer0248) 維護。  
沒有東西可以寫了。
