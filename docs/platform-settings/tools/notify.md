# 斗內通知

## 通知設定

::: warning
- 當這頁有變更的時候，請記得一定要重新整理 OBS 上的瀏覽器。
- Twitch 機器人通知並不是所有使用者都可以設定。
:::

您可以在這邊變更 OBS 通知的樣式、格式、文字大小、邊框、顏色、顯示多久、進場動畫等。

## 通知自訂 CSS 設定

可以自由調整通知跳出來的顏色及文字大小。

## 禁詞設定

誰知道會不會有黑粉呢。  
只要有包含在禁詞中的字，斗內就不會被成功送出。  
平台有提供預設禁詞，不定期更新，並且會強制刪除。
