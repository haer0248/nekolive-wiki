斗內列表樣式

可使用參數

參數設定預設值說明
limit整數99限制要顯示多少則斗內訊息
horziedtrue (是), false (否)false以水平方式顯示斗內列表
size整數300設定在水平模式下，單個斗內訊息的寬度
（建議設定 200 以上）
no-newlinetrue (是), false (否)false超出寬度時，文字最後方顯示 ...
keep-boxtrue (是), false (否)false沒有留言時下方保留空白
no-boxtrue (是), false (否)false不顯示留言
hide-avatartrue (是), false (否)false隱藏頭貼
no-commatrue (是), false (否)false不顯示格式化千分位

參數使用教學

範例
Widgets/樣式名稱/您的 UUID?limit=3&horzied=true
沒有問號
Widgets/樣式名稱/您的 UUID?limit=3
有問號
Widgets/樣式名稱/您的 UUID?limit=3&hide-avatar=true
有問號、有連結號
Widgets/樣式名稱/您的 UUID?limit=3&horzied=true&size=300
